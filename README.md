## Getting Started

To get a local copy up and running follow these simple example steps.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AndrewCheung360/Portfolio-Website-v0.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   
3. Run the development server
   ```sh
   npm run dev
   ```
